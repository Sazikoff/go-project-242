## Requirements

* Go 1.19
* Make
* [golangci-lint](https://golangci-lint.run)

## Setup

```bash
git clone git@github.com:Sazikoff/go-project-242.git
cd go-project-242
make install
```

## Run linter

```bash
make lint
```

## Run tests

```bash
make test
```
